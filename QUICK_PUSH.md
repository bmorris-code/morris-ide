# 🚀 Quick Push Commands

After you create the GitHub repository at https://github.com/bmorris-code/morris-ide, run these commands:

```bash
git push -u origin master
git push origin --tags
```

That's it! GitHub Actions will automatically build all platforms:

- 🪟 Windows: MorrisIDE-1.0.0-Setup.exe
- 🐧 Linux: MorrisIDE-1.0.0.AppImage  
- 🍎 macOS: MorrisIDE-1.0.0.dmg
