# Download Morris IDE

🤖 **AI-Native Secure Development Environment**

---

## 🎯 Choose Your Platform

### 🪟 Windows
**Download:** [MorrisIDE-1.0.0-Setup.exe](https://github.com/morris-ide/morris-ide/releases/latest/download/MorrisIDE-1.0.0-Setup.exe)

**Installation:**
1. Download the `.exe` file
2. Double-click to run the installer
3. Follow the setup wizard
4. Launch from desktop shortcut

**Requirements:** Windows 10 or later

---

### 🐧 Linux
**Download:** [MorrisIDE-1.0.0.AppImage](https://github.com/morris-ide/morris-ide/releases/latest/download/MorrisIDE-1.0.0.AppImage)

**Installation:**
```bash
# Make executable
chmod +x MorrisIDE-1.0.0.AppImage

# Run the application
./MorrisIDE-1.0.0.AppImage
```

**Requirements:** Ubuntu 18.04+, Fedora 30+, or equivalent

---

### 🍎 macOS
**Download:** [MorrisIDE-1.0.0.dmg](https://github.com/morris-ide/morris-ide/releases/latest/download/MorrisIDE-1.0.0.dmg)

**Installation:**
1. Download the `.dmg` file
2. Double-click to open
3. Drag Morris IDE to Applications folder
4. Launch from Launchpad or Applications

**Requirements:** macOS 10.15 (Catalina) or later

---

## 🛠️ After Installation

### 1. Configure AI Provider
Open Morris IDE and click the **Settings** icon (⚙️):

- **Groq** (Free): Get API key at https://console.groq.com
- **Moonshot AI**: Get API key at https://platform.moonshot.cn  
- **OpenAI**: Get API key at https://platform.openai.com

### 2. Start Coding!
The AI assistant will help you with:
- ✅ Code explanations
- 🐛 Bug detection and fixes
- 🔧 Code refactoring
- 📚 Documentation generation
- ⚡ Performance optimizations
